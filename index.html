プライバシーポリシー

最終更新日: 2026年4月17日

DropNow（以下「本アプリ」）は、ユーザーのプライバシーを尊重し、個人情報の保護に努めます。本プライバシーポリシーは、本アプリにおける情報の取り扱いについて説明します。

1. 収集する情報

本アプリでは、以下の情報を収集します。

  • プッシュ通知トークン: プッシュ通知を送信するために、Expo Push Token を収集します。これは端末を識別するためのトークンであり、個人を特定する情報ではありません。
  • 通知設定: カテゴリごとの通知ON/OFF設定を、プッシュ通知トークンと紐づけてサーバーに保存します。

2. 収集しない情報

本アプリでは、以下の情報は収集しません。

  • 氏名、メールアドレス、電話番号などの個人情報
  • 位置情報
  • 端末内のお気に入りデータ（端末内にのみ保存され、サーバーに送信されません）

3. 情報の利用目的

収集した情報は、以下の目的で利用します。

  • ユーザーが有効にしたカテゴリの新着情報をプッシュ通知でお届けするため

4. 情報の第三者提供

本アプリで収集した情報を第三者に提供することはありません。ただし、プッシュ通知の配信にあたり、Expo（Expo Push Notification Service）を利用しています。

5. データの保管

プッシュ通知トークンおよび通知設定は、Cloudflare D1 データベースに保管されます。

6. データの削除

アプリを削除すると、端末内のデータ（お気に入り、通知設定）は自動的に削除されます。サーバーに保存されたプッシュ通知トークンは、一定期間使用されない場合に自動的に削除されます。
